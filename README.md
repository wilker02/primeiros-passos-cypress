# 🚀 My Cypress Automation

This project was developed using **Cypress** to automate end-to-end tests for key system features, focusing on the **login flow** and **user information update process**.

Cypress is a modern JavaScript-based testing framework that allows developers and QA engineers to create reliable automated tests and run them directly in a real browser.

---

## 📌 Features Tested

- ✅ User login with valid credentials  
- ✅ Login validation  
- ✅ User information update  
- ✅ Success message validation  
- ✅ Navigation and page access checks  

---

## 🛠️ Technologies Used

- **Cypress**
- **JavaScript**
- **Node.js**
